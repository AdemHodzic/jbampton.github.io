Installing [rbenv](https://github.com/rbenv/rbenv){:target="_blank"} is easy with [Homebrew](http://brew.sh/){:target="_blank"} with the next two commands:

```
$ brew update
$ brew install rbenv
```

Then to install the latest current version of Ruby 2.4 type:

```
$ rbenv install 2.4.0
```
